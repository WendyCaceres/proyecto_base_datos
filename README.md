# proyecto_base_datos

## ⚙️ Tecnologías

- PostgreSQL 15
- MongoDB
- Redis
- Docker & Docker Compose
- Python (Faker)
- Node.js

## 🚀 Inicio Rápido

1. **Clonar el repositorio**
```bash
git clone <repo-url>
cd proyecto-base-datos
```

3. **Levantar servicios**
```bash
docker-compose up -d
```

4. **Inicializar base de datos**
```bash
./init_data.bat
```
